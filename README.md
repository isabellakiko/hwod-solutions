# 华为OD机试算法题库

> 212道真题 + Java/Python/JS多语言解答 + 在线复习系统

## 在线访问

https://isabellakiko.github.io/hwod-solutions

## 功能特点

- 212道华为OD机试真题
- 支持 Java、Python、JavaScript 多语言解答
- 按分值筛选（100分/200分）
- 按考试类型筛选（A卷/B卷）
- 深色/浅色主题切换
- 代码高亮显示

## 题目结构

每道题包含：
- 题目描述
- 输入/输出说明
- 示例用例
- 解题思路
- 多语言代码实现

## 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- React 19
- Vite 7
- React Router 7
- Prism.js (代码高亮)
