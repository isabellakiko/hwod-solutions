## 数据结构设计

### 题目数据

```typescript
interface Problem {
  id: string;           // 题目 ID
  title: string;        // 题目标题
  examType: 'A' | 'B';  // 考试类型
  score: 100 | 200;     // 分值
  description: string;  // 题目描述
  inputDesc: string;    // 输入说明
  outputDesc: string;   // 输出说明
  examples: Example[];  // 示例
  solution: string;     // 解题思路
  codes: Codes;         // 代码
}

interface Example {
  input: string;
  output: string;
  explanation: string;
}

interface Codes {
  java: string;
  python: string;
  javascript: string;
  cpp: string;
  c: string;
}
```

## 交互设计

### 筛选交互
- 单选筛选，点击即生效
- 列表实时更新
- 保持滚动位置

### 语言切换
- Tab 形式切换
- 记住上次选择
- 平滑过渡动画

### 主题切换
- 点击切换深色/浅色
- 保存到 localStorage
- 跟随系统设置（可选）
