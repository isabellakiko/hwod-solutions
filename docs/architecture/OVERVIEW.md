## 技术选型

| 层级 | 技术 | 选型理由 |
|------|------|----------|
| 前端框架 | React 19 | 生态成熟，组件化开发 |
| 构建工具 | Vite 7 | 快速启动，HMR 体验好 |
| 路由 | React Router 7 | 官方推荐，功能完整 |
| 代码高亮 | Prism.js | 轻量，支持多语言 |
| 部署 | GitHub Pages | 免费，与仓库集成 |

## 数据流

```
用户访问
    ↓
React Router 匹配路由
    ↓
页面组件加载
    ↓
import() 动态导入 JSON
    ↓
组件渲染数据
    ↓
用户交互（筛选/切换）
    ↓
状态更新 → 重新渲染
```

## 目录结构设计

```
src/
├── components/           # 可复用 UI 组件
│   ├── Layout/           # 布局相关
│   ├── Problem/          # 业务组件
│   └── FormattedText/    # 通用组件
├── pages/                # 页面级组件
├── data/                 # 静态数据
│   └── problems/         # 题目 JSON（按 ID 分文件）
├── hooks/                # 自定义 Hooks
└── styles/               # 全局样式
```

**设计原则**:
- 组件按职责分类
- 数据与视图分离
- 单一文件单一职责
